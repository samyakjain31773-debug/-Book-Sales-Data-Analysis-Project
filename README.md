# -Book-Sales-Data-Analysis-Project
This project focuses on analyzing a dataset of book sales using Python. 
It includes data cleaning, exploration, visualization, and insights extraction 
to understand trends in book publishing, genres, ratings, and sales performance.


🔗 Dataset Source
Dataset used in this project is from Kaggle:
👉 https://www.kaggle.com/datasets/thedevastator/books-sales-and-ratings
  

🚀 Features
Data Cleaning & Preprocessing
Handling Missing Values
Duplicate Data Removal
Exploratory Data Analysis (EDA)
Data Visualization using Matplotlib & Seaborn
Statistical Summary of Dataset
Insights on:
Genre distribution
Language distribution
Author performance
Publisher revenue
Sales trends

🛠️ Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📂 Dataset Information
The dataset contains the following columns:
Publishing Year
Book Name
Author
Language Code
Author Rating
Book Average Rating
Book Ratings Count
Genre
Gross Sales
Publisher Revenue
Sale Price
Sales Rank
Publisher
Units Sold

🔍 Data Cleaning Steps
Removed books published before 1900
Handled missing values using dropna()
Checked null values using isna().sum()
Removed duplicates
Cleaned specific columns (e.g., Book Name)

📊 Exploratory Data Analysis
1. Genre Distribution
Bar chart showing number of books per genre
Fiction dominates dataset
2. Language Distribution
Pie chart showing percentage of books in each language
English is the most common language
3. Author Analysis
Average rating per author
Top & bottom authors based on ratings
4. Publisher Analysis
Total revenue generated by each publisher
Sorted highest to lowest
5. Sales Analysis
Scatter plot of Sale Price vs Units Sold
Box plot of Ratings Count across Genres

📈 Key Insights
Fiction genre has the highest number of books
English language dominates the dataset
Some authors consistently have higher ratings
Publisher revenue varies significantly
No strong direct relationship between price and units sold
